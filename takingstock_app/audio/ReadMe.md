# AUDIO FILES FOR TAKING STOCK INSTALL
### Last edited by Tench C 05.26.26

Hello future person! This is the folder where you place audio files to be used in the taking stock install. They should be named based off their cluster number, which will be used to find the matching audio file to the key video file per arrangement. More info on arrangements can be found in bin/data/configREADME.md

The audio file is found using the cluster_no column of the installation.csv file found in the videos folder.
