# Taking Stock Install Repo
### Last updated by Tench C 04.19.26

## Intro:

Hello future person! this repo is built to generate different layouts of videos created from the original takingstock repo: https://github.com/mandiberg/takingstock

It is mostly plug and play, however it does need a few tweaks to get set up. Follow these directions.

**1.** Inside the openframeworks folder there is a readme with directions on which version of openframeworks to download and how to set that up. THIS REPO DOES NOT WORK WITH THE DEFAULT OPENFRAMEWORKS VERSION! It needs some file replacements and customization.

**2.** Videos need to be placed into the videos folder, alongside a csv file called installation.csv in the videos folder. That csv should have the following columns: [cluster_no, hsv_no, pose_no, width, height, ratio, file_name, duration]. The csv is generated by running make_video.py and merge_expanded_videos.py in the takingstock repo. 

**3.** open the terminal and run the following commands:
    **3a.** cd (path to this folder)
    **3b.** make
    **3c.** make RunRelease

The first time you run it you will have to run the make command, but every additional time you can just run makeRunRelease.

## Config:

This app has many config options in terms of how videos are processed and developed. all config can be altered in the file bin/data/config.txt
